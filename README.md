# 📝 Task Manager (React App)

A simple and responsive Task Manager (To-Do Application) built using React.js. This app allows users to manage their daily tasks efficiently.

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🎨 Clean and responsive UI
- ⚡ Built using React Hooks (useState)

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS
